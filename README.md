# Easymed - Healthcare Management System

Easymed is a Spring Boot healthcare management system for managing patients, doctors, appointments and prescriptions.

## Features
- User Registration & Login
- Doctor & Patient Management
- Appointment Booking
- Prescription Management
- Lab Report Upload

## Teck Stack
- Spring Boot
- Thymeleaf
- MySQL
- Maven
- Spring Security

## How to Run
1. Colne the repository
2. Create MySQL database : EasyMed
3. Update application.propeties
4. Run Using :
   mvn spring-boot:run
